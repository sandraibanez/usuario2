El curso de 2º DAW de Desarrollo de Aplicaciones Web es un curso muy completo que te prepara para trabajar como desarrollador web. En este curso aprenderás todo lo necesario para crear aplicaciones web, desde el diseño de la interfaz hasta el desarrollo del backend.

    Las asignaturas se dibiden en cuatro bloques

  • Desarrollo web en entorno cliente:

    En este bloque aprenderás a crear la interfaz de tu aplicación web utilizando HTML, CSS y JavaScript, que son algunas de los lenguajes que aprenden en este curso.

  • Desarrollo web en entorno servidor:

    En este bloque aprenderás a desarrollar el backend de tu aplicación web utilizando PHP, Java o Python, que son algunas de los lenguajes que aprenden en este curso.

  • Desplige de aplicaciones web:

    En este bloque aprenderas a lanzanzar tus aplicaciones web con herramientas como github.

  • Diseño de aplicaciones web:

    En este bloque aprenderas a deseñar el diseño de la pagina web con html y css.
